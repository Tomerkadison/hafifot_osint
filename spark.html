קורס Spark מעשי · עברית

תרגילים ב-Apache Spark

המשך ישיר לקורסי ה-pandas וה-Polars — אבל בלי לחזור על אותם דברים. כאן מתמקדים במה שמייחד את Spark: חישוב מבוזר ועצל, פונקציות חלון, SQL, מידע מקונן, וביצועים — בדיוק כמו שעובדים ב-Palantir Foundry.

הפילוסופיה: הבסיס (read / select / filter / groupBy) — בקצרה, כי אתם כבר מכירים אותו מ-pandas ו-Polars. כל השאר עוסק במה ש-Spark עושה אחרת: lazy execution, shuffles, Window functions, JSON מקונן, ו-UDFs מול פונקציות מובנות.

התקנה

ל-Spark צריך Java (פעם אחת) ואז pip install pyspark. המדריך המלא — כולל אופציות של Google Colab ו-Docker למי שלא רוצה להתקין — נמצא בקובץ SETUP.md. 

brew install openjdk@17        # macOS (או Temurin 17 ב-Windows)
pip install "pyspark==3.5.1" jupyter pandas pyarrow

הכי קל להתחיל? פתחו notebook ב-Google Colab והריצו !pip install pyspark — שם Java כבר מותקן.
