פרק 10 · הפרויקט המסכם

בניית דוח GDP מקצה לקצה

משימה רציפה: ניקוי, חיבור, חישוב, וכתיבה — כמו pipeline אמיתי.

1על מה התרגיל?

הפרויקט המסכם (22 שאלות, ארוך). זו משימה אחת רציפה: בונים דוח אנליטי שלם משלושה מקורות נתונים אמיתיים. כל שאלה ממשיכה את הקודמת — בדיוק כמו כתיבת טרנספורמציה אמיתית מתחילתה ועד סופה, כולל כתיבת התוצאות לקובץ.

💡 זה תרגיל רציף: המשתנה gdp גדל ומתעשר משאלה לשאלה. הריצו את התאים לפי הסדר.

הוצאות מו"פ (R&D) לפי מקור מימון, 1996–2016. נשתמש בעמודת Government.

6מה תתרגלו (הכל ביחד!)

  • קריאה, rename, describe וחקירה
  • window functions: over + arg_min/arg_max + get
  • pivot ו-max_horizontal
  • טיפול בבעיית זיהוי טיפוס + cast
  • join על שני וגם שלושה מפתחות
  • חישוב מדדים: תוצר לנפש, מו"פ כאחוז מהתוצר
  • write_csv לשמירת התוצרים

7איך מתחילים

  1. ודאו ש-Polars מותקן: pip install polars (גרסה 1.24.0).
  2. ודאו ששלושת קובצי ה-CSV נמצאים בתיקיית data/.
  3. פתחו את Exercises.ipynb והריצו את התאים לפי הסדר.
  4. בסוף ייווצר תיקיית processed/ עם שלושה דוחות.
  5. סיימתם? השוו מול Solutions.ipynb — וסיימתם את הקורס! 🎉
